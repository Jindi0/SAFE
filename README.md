# SAFE
This code implements [SAFE: Similarity-Aware Multi-modal Fake News Detection](https://link.springer.com/chapter/10.1007/978-3-030-47436-2_27) model.


## Requirements
- Python 3.7
- TensorFlow 2.2

## Getting Started

### Install requirements
```
pip install -r requirements.txt
```

### Train
```
python3 train.py
```


### Test
```
python3 test.py
```
